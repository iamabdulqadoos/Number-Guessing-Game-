# 🎮 Number Guessing Game (Python)

A feature-rich console-based Number Guessing Game built in Python.  
This project includes player authentication, history tracking, leaderboard, progress analytics, and a review system.

## 🚀 Features

- 👤 Player Login System (New & Returning users)
- 🎯 Difficulty Levels (Easy, Medium, Hard)
- ⏱️ Time tracking for each game
- 🔢 Attempt tracking system
- 🏆 Global Leaderboard (Top players)
- 📊 Personal Profile System
- 📈 Progress analytics (Best, Worst, Average performance)
- 🕘 Game history tracking
- 💬 User review system

---

## 🛠️ Technologies Used

- Python 3
- File Handling (TXT storage system)
- Random module
- Time module
- Datetime module
